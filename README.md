# Slack Enhancer Extension

Slack の使い勝手を向上させる Chrome 拡張機能です。
チームやプロジェクトでのコミュニケーションをよりスムーズにするための補助機能を提供します。

## ✨ 主な機能

### 1. 返信（スレッド）の強調表示
Slack 上で返信がついているメッセージ（「X件の返信」などの表示）を見やすくハイライトします。
これにより、重要なスレッドでの議論や返信の見落としを防ぎ、コミュニケーションのスピードを上げることができます。
- **特徴**: 返信ボタンが赤枠と薄い赤色の背景で強調され、ホバー時にアニメーションします。

### 2. Atlassian リンク（Jira / Confluence）のタイトル自動変換
Jira の課題リンクや Confluence のページリンクが Slack に貼り付けられた際、無機質な URL 文字列を自動的に「ページのタイトル」や「課題キー + サマリー」に置き換えます。
- **Before**: `https://mycompany.atlassian.net/browse/PROJ-123`
- **After**: `PROJ-123 ログイン画面のレイアウト崩れを修正`
- **仕組み**: ブラウザのログインセッション（Cookie）を利用して情報を取得するため、これまで通りブラウザで Jira/Confluence にログインしていれば特別な設定なしで動作します。

## 📦 インストール方法（開発者モード）

この拡張機能は Chrome Web Store には公開されていません。以下の手順でローカル環境にインストールしてください。

1. **リポジトリの取得**
   このリポジトリをローカルにクローン（または ZIP でダウンロードして解凍）してください。
   ```bash
   git clone https://github.com/shinyanemoto/slack-enhancer-extension.git
   ```

2. **Chrome 拡張機能管理画面を開く**
   Chrome ブラウザのアドレスバーに以下を入力して開きます。
   `chrome://extensions/`

3. **デベロッパー モードを有効化**
   画面右上にある「デベロッパー モード」のスイッチを **ON** にします。

4. **拡張機能を読み込む**
   左上に現れる「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、**手順 1 で取得したフォルダ（`manifest.json` が入っているフォルダ）** を選択します。

5. **完了**
   拡張機能リストに "Slack Enhancer" が追加されればインストール完了です。Slack のページをリロードして動作を確認してください。

## 🛠 技術情報
- **Manifest Version**: V3
- **Permissions**:
  - `activeTab`, `scripting`: ページ内容の操作用
  - `host_permissions`: `app.slack.com` およびリンク先情報の取得用（`<all_urls>`）
- **主要構成**:
  - `content.js`: Slack 上の DOM を監視 (`MutationObserver`) し、要素の書き換えを担当。
  - `background.js`: クロスオリジンでのデータ取得（Jira/Confluence API への fetch）を担当。

## ⚠️ 注意事項
- **Jira/Confluence のタイトル取得について**:
  ご利用のブラウザで対象のサービスにログインしている必要があります。ログインしていない場合、タイトル情報は取得できず URL のまま表示されます。
- **Slack の仕様変更について**:
  Slack の DOM 構造（クラス名など）は頻繁に変更される可能性があります。もし機能しなくなった場合は、Issue でお知らせいただくか、スタイルセレクタの修正をご検討ください。
